Apriori Algorithm Practice Questions

Try to solve the following problems using the Apriori algorithm.

Question 1

Given the following transaction data:

  • Transaction 1: {milk, bread, butter}
  • Transaction 2: {bread, butter}
  • Transaction 3: {milk, bread, diaper}
  • Transaction 4: {milk, bread, butter, diaper}
  • Transaction 5: {milk, diaper}

Determine itemsets that meet the minimum support of 2.

Discussion

Solution steps:

Identify the itemsets in each transaction.li>
  • Compute support for each itemset.
  • Select itemsets that meet the minimum support of 2.

    • Eligible itemsets:

    • Milk: 4
    • Bread: 4
    • Butter: 2
    • Diaper: 3

    Explanation: Itemsets with support less than 2 are discarded because they do not meet the minimum support criteria. For example, "Butter" only appears in 2 transactions, so it exactly meets the minimum support criteria.

    Question 2

    Given the same transaction data as in Question 1. Calculate the association rules with a minimum confidence of 50%.

    Discussion

    Solution steps:

    1. Use itemsets that have met the minimum support from Question 1.
    2. Calculate the confidence for each pair of itemsets.
    3. Choose a pair that has confidence ≥ 50%.

    Example results:

    • Milk -> Bread: 75%
    • Diaper -> Milk: 66%

    Explanation: Confidence is calculated using the formula: \[ \text{Confidence(A → B)} = \frac{\text{Support(A ∪ B)}}{\text{Support(A)}} \] For example, for the rule{Milk} -> {Bread}: - Support(Milk ∪ Bread) is 3 (Milk and Bread appear together in 3 transactions). - Support(Milk) is 4. Confidence = \( \frac{3}{4} = 75\% \). This rule meets the minimum confidence requirement of 50%.

    Question 3

    Given transaction data:

    • Transaction 1: {pen, notebook}
    • Transaction 2: {pen, notebook, eraser}
    • Transaction 3: {notebook, eraser}
    • Transaction 4: {pen, eraser}
    • Transaction 5: {notebook}

    Determine frequent itemsets with minimum support of 2, then calculate association rules with confidence ≥ 60%.

    Discussion

    Solution steps:

    1. Identify all itemsets from transaction data.
    2. Calculate support for each itemset.
    3. Select itemsets that meet the support requirements.
    4. Calculate confidence for the association rules formed.

    Results:

    • Frequent itemsets: {pen}, {notebook}, {eraser}
    • Example association rule: {pen} -> {notebook}: 66.6%

    Explanation: Minimum support is met if the item appears in at least 2 transactions. Confidence is calculated as in Problem 2. For example, the rule {pen} -> {notebook}: - Support(pen ∪ notebook) is 2. - Support(pen) is 3. Confidence = \( \frac{2}{3} = 66.6\% \), meets the minimum confidence requirement.

    Question 4

    Provide an interpretation for the following association rules:

    • If someone buys Milk, then the probability of buying Bread is 75%.
    • If someone buys Diapers, then the probability of buying Milk is 66%.

    Discussion

    Interpretation:

    • The first rule shows a strong relationship between Milk and Bread. Promotion of one product can increase purchases of the other product.
    • The second rule shows that Diaper and Milk have a significant relationship. This can be used for marketing strategies such as product combination discounts.

    Explanation: This interpretation helps in business decision making, such as designing promotions based on consumer purchasing patterns.
