Implementation Example in Python

Here is a simple example of the Apriori algorithm in Python: 

  from itertools import combinations

  # Data transaksi
  transaksi = [
      {'milk', 'bread'},
      {'milk', 'diaper', 'beer', 'bread'},
      {'milk', 'diaper', 'beer', 'coke'},
      {'bread', 'milk', 'diaper', 'beer'},
      {'bread', 'milk', 'diaper', 'coke'},
  ]

  # Ambang minimum support
  min_support = 2

  # Fungsi untuk menghitung support
  def calculate_support(itemset, transaksi):
      return sum(1 for transaction in transaksi if itemset.issubset(transaction))

  # Algoritma Apriori
  def apriori(transaksi, min_support):
      # Mengumpulkan semua item unik dari transaksi
      items = set(item for transaction in transaksi for item in transaction)
      frequent_itemsets = []
      k = 1  # Panjang itemset
      
      # Inisialisasi dengan item tunggal
      current_itemsets = [{item} for item in items]
      
      while current_itemsets:
          # Hitung support untuk setiap itemset
          valid_itemsets = []
          unique_itemsets = set(frozenset(itemset) for itemset in current_itemsets)
          for itemset in unique_itemsets:
              support = calculate_support(itemset, transaksi)
              if support >= min_support:
                  frequent_itemsets.append((itemset, support))
                  valid_itemsets.append(itemset)
          
          # Buat kombinasi itemset berikutnya
          k += 1
          current_itemsets = [
              set1 | set2
              for i, set1 in enumerate(valid_itemsets)
              for set2 in valid_itemsets[i+1:]
              if len(set1 | set2) == k
          ]
      
      return frequent_itemsets

  # Menjalankan algoritma
  hasil_itemsets = apriori(transaksi, min_support)

  # Tampilkan hasil
  print("Frequent Itemsets:")
  for itemset, support in hasil_itemsets:
      print(f"Itemset: {set(itemset)}, Support: {support}")

The above example searches for itemsets that meet a certain minimum support. You can try this code in a Python environment such as Jupyter Notebook or Google Colab.

Implementation Example in PHP

Here is a simple example of the Apriori algorithm in PHP:

<?php
// Data transaksi
$transaksi = [
    ['milk', 'bread'],
    ['milk', 'diaper', 'beer', 'bread'],
    ['milk', 'diaper', 'beer', 'coke'],
    ['bread', 'milk', 'diaper', 'beer'],
    ['bread', 'milk', 'diaper', 'coke']
];

// Ambang minimum support
$min_support = 2;

// Fungsi Apriori
function apriori($transaksi, $min_support) {
    $itemset = [];
    foreach ($transaksi as $items) {
        foreach ($items as $item) {
            $itemset[$item] = ($itemset[$item] ?? 0) + 1;
        }
    }
    
    $result = [];
    foreach ($itemset as $item => $support) {
        if ($support >= $min_support) {
            $result[] = [$item, $support];
        }
    }
    return $result;
}

print_r(apriori($transaksi, $min_support));
?>

The PHP code above displays itemsets that meet the minimum support specified. You can run this code on a local server to test the results.
